Carpeta de logos (PNG recomendado)
==================================

1. Guarda aquí tus archivos PNG (fondo transparente si puede ser).

2. Nombres por defecto que usa ANDATALAB.html:
   logo-01.png
   logo-02.png
   logo-03.png
   logo-04.png
   logo-05.png
   logo-06.png

3. Si usas otros nombres, abre ANDATALAB.html y en el array BRANDS
   cambia cada ruta `img:'logos/...'` y el campo `name` (texto si la imagen no carga).

4. Altura visual en web: ~22–30px; anchos razonables hasta ~140px.
